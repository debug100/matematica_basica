📘 2. Conjuntos – Guia Completo e Intuitivo
🔹 1. Definições Básicas
🔸 O que é um conjunto?

Um conjunto é uma coleção de elementos.

Exemplo:

A = {1, 2, 3}
B = {a, b, c}
🔸 Pertinência (∈)

Indica se um elemento pertence ao conjunto:

1 ∈ A → (1 pertence a A)
4 ∉ A → (4 não pertence a A)
🔸 Inclusão (⊆)

Indica se um conjunto está contido em outro:

A ⊆ B → todo elemento de A está em B

Exemplo:

A = {1,2}
B = {1,2,3}
👉 A ⊆ B
🔸 Conjuntos Numéricos Importantes
ℕ → naturais (0,1,2,3...)
ℤ → inteiros (...,-1,0,1...)
ℚ → racionais (frações)
ℝ → reais (todos os números)
🔹 2. O Poder dos Conjuntos
🔸 Conjunto das Partes (𝒫(A))

É o conjunto de todos os subconjuntos de A.

Exemplo:

A = {1,2}

𝒫(A) = { ∅, {1}, {2}, {1,2} }

🔸 Cardinalidade

É a quantidade de elementos.

|A| = número de elementos de A

👉 Fórmula importante:

Se |A| = n
então |𝒫(A)| = 2ⁿ

Exemplo:

A = {1,2,3} → 3 elementos
𝒫(A) terá 2³ = 8 subconjuntos
🔹 3. Operações e Álgebra de Conjuntos
🔸 União (∪)

Elementos que estão em A ou em B

A ∪ B

Ex:

{1,2} ∪ {2,3} = {1,2,3}
🔸 Interseção (∩)

Elementos comuns

A ∩ B

Ex:

{1,2} ∩ {2,3} = {2}
🔸 Diferença (−)

Elementos que estão em A mas não em B

A − B

Ex:

{1,2,3} − {2} = {1,3}
🔸 Complementar

Elementos que não estão no conjunto (em relação a um universo U)

Aᶜ
🔸 Leis importantes
Comutativa:
A ∪ B = B ∪ A
A ∩ B = B ∩ A
Distributiva:
A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)
Leis de De Morgan:
(A ∪ B)ᶜ = Aᶜ ∩ Bᶜ
(A ∩ B)ᶜ = Aᶜ ∪ Bᶜ

👉 Essas leis permitem provar identidades.

🔹 4. Estruturas de Produto
🔸 Produto Cartesiano (A × B)

É o conjunto de pares ordenados:

A × B = {(a,b)}

Exemplo:

A = {1,2}
B = {x,y}

👉 A × B =
{(1,x), (1,y), (2,x), (2,y)}

🔸 Importante:
(1,2) ≠ (2,1)
👉 A ordem importa!
🔸 Tamanho do produto

Se:

|A| = m
|B| = n

Então:

|A × B| = m × n
🔹 5. Partição de Conjuntos
🔸 O que é partição?

É dividir um conjunto em partes sem sobreposição que cobrem tudo.

Regras:

Nenhum subconjunto é vazio
Não há interseção entre eles
A união de todos dá o conjunto original
🔸 Exemplo:

A = {1,2,3,4}

Uma partição possível:

{1,2} e {3,4}

Outra:

{1}, {2}, {3}, {4}
🔸 Ideia central:

👉 Você está “quebrando” o conjunto em pedaços organizados, sem repetir elementos.
