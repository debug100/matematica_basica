📘 3. Funções – Guia Completo e Intuitivo
🔹 1. Definição Formal
🔸 O que é uma função?

Uma função é um tipo especial de relação onde:

👉 cada elemento do domínio está associado a exatamente um único valor

🔸 Estrutura de uma função

Uma função f: A → B tem:

Domínio (A) → de onde saem os valores
Contradomínio (B) → onde os valores chegam
Imagem → os valores que realmente aparecem
🔸 Exemplo

f(x) = x², com domínio ℝ

f(2) = 4
f(3) = 9

👉 Domínio: ℝ
👉 Contradomínio: ℝ
👉 Imagem: números ≥ 0

🔸 Regra fundamental

⚠️ Um elemento do domínio NÃO pode ter dois resultados diferentes.

✔ válido:

f(2) = 4

❌ inválido:

f(2) = 4 e f(2) = 5
🔹 2. Classificação das Funções
🔸 Função Injetora (um-para-um)

👉 Elementos diferentes → resultados diferentes

Se:

f(a) = f(b) → então a = b

Exemplo:

f(x) = x + 1

✔ nunca repete valores

🔸 Função Sobrejetora

👉 A imagem é igual ao contradomínio

Ou seja:

Todo elemento do contradomínio é atingido

Exemplo:

f(x) = x³ (em ℝ → ℝ)
🔸 Função Bijetora

👉 É injetora + sobrejetora

Não repete valores
Cobre todo o contradomínio

👉 É a mais importante!

✔ Por quê?

Permite função inversa
Base de áreas como criptografia e compressão
🔹 3. Operações com Funções
🔸 Composição de Funções

É aplicar uma função dentro da outra:

(f ∘ g)(x) = f(g(x))
🔸 Exemplo

f(x) = x + 1
g(x) = 2x

(f ∘ g)(x) = f(2x) = 2x + 1

(g ∘ f)(x) = g(x + 1) = 2(x + 1) = 2x + 2

⚠️ Importante:
👉 f ∘ g ≠ g ∘ f (na maioria dos casos)

🔸 Função Inversa

A inversa “desfaz” a função:

Se:

f(x) = y

Então:

f⁻¹(y) = x
🔸 Condição para existir inversa

👉 A função precisa ser bijetora

🔸 Exemplo

f(x) = 2x + 3

Passos:

y = 2x + 3
Troca x ↔ y
Resolve:

x = 2y + 3
y = (x - 3)/2

👉 f⁻¹(x) = (x - 3)/2
